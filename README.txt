# 🪨 Rock-Paper-Scissors Game (Java)

This is a simple command-line **Rock-Paper-Scissors** game written in Java. The game allows a user to play against the computer as many times as they like.

## 📁 File

- `Methods.java` — Contains the full game logic.

## 🚀 How It Works

1. The computer randomly chooses between `rock`, `paper`, and `scissors`.
2. The user is prompted to enter their choice.
3. The result is displayed based on standard game rules.
4. The user can choose to play again or exit.
